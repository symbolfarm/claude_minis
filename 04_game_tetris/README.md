# Tetris Game

A classic Tetris game implemented using Pygame.

## Installation

```bash
pip install -r requirements.txt
```

## Running the Game

```bash
python tetris.py
```

## Controls

- Left Arrow: Move piece left
- Right Arrow: Move piece right
- Down Arrow: Move piece down faster
- Up Arrow: Rotate piece
- Space: Hard drop
- P: Pause game
- Esc: Quit game
